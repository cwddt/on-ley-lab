# on-ley-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本项目主要用于存放和展示各类独立的 HTML 页面。页面内容涵盖文档、工具、演示等不同用途，每个页面均为独立文件，不依赖特定后端服务。

仓库本身不针对任何单一网站或域名，仅作为页面归档与发布的静态空间。

## 目录说明

```
/
├── pages/           # 存放各 HTML 页面的主目录
│   ├── example1/    # 示例页面1 相关文件
│   ├── example2/    # 示例页面2 相关文件
│   └── ...
├── assets/          # 共享资源文件（CSS、JS、图片等）
├── index.html       # 仓库入口页（可选）
└── README.md        # 本文件
```

- `pages/` 下每个子目录对应一个独立页面，包含其 HTML、样式及脚本。
- `assets/` 用于存放多个页面可复用的公共资源。
- `index.html` 可作为仓库的导航入口，非必须。

## 页面归档说明

- 所有页面均为纯静态 HTML，可直接在浏览器中打开。
- 页面之间无硬性依赖，可单独使用或通过链接跳转。
- 归档的页面可能不定期更新或新增，旧页面保留但不保证持续维护。
- 页面内容不涉及具体商业、营销或推广信息，仅作为功能性或演示性展示。

## 维护说明

- 欢迎通过 Issue 或 Pull Request 提交改进建议。
- 新增页面时，请在 `pages/` 下创建独立目录，并保持目录结构清晰。
- 共享资源请优先放置于 `assets/`，避免各页面重复加载。
- 所有提交需遵守开源社区规范，不包含侵权、恶意或违法内容。
